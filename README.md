<h1 align="center"> Jogo do número secreto </h1>
</h2>Sobre meu projeto:</h2>
</p>Projeto do jogo do número secreto, utilizando logica de programação</p>

✔️ Técnicas e tecnologias utilizadas
- `JavaScript`
- `Css`
- `Html`

# :hammer: Funcionalidades do projeto
- `Funcionalidade 1`: Gerar um numero secreto de 1 ao limite maximo, e guardar em um array, com uma função para os numeros nao se repetirem.
- `Funcionalidade 2`: Verificar chute colocado no input, e se for correto, alterar o HTML, e se for errado tambem alterar o HTML
- `Funcionalidade 3`: Função para dar dicas se o número chutado é menor ou maior que o número secreto.
- `Funcionalidade 4`: Reiniciar jogo, ao acertar o número secreto, o botão "jogo novo" é ativado, e ao clicar cria outro jogo.
